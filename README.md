# 📝 To-Do List Web Application

## 📌 Overview
The To-Do List is a simple and interactive web application designed to help users manage their daily tasks efficiently. It allows users to add, delete, and track tasks with a clean and user-friendly interface.

---

## 🚀 Features

### 🔹 Task Management
- Add new tasks  
- Delete tasks  
- Mark tasks as completed  
- Track pending and completed tasks  

### 🔹 User Interface
- Clean and minimal design  
- Interactive task list  
- Responsive layout for mobile and desktop  
- Smooth user experience  

---

## 🛠️ Technologies Used
- HTML5 – Structure  
- CSS3 – Styling and layout  
- JavaScript – Functionality and task handling
